# 🌐 GEOSIGHT — Live Geopolitics Dashboard

A real-time geopolitics intelligence dashboard powered by AI.

## 🔴 Live Site
👉 [geosight-dashboard.netlify.app](https://geosight-global.netlify.app)

## 📋 Features
- **Global Overview** — AI-generated highlights of major geopolitical events
- **Active Conflicts** — Live war updates with severity levels
- **Conflict Map** — Interactive world map with conflict zone markers
- **War-Affected Markets** — Commodity prices affected by global conflicts
- **Breaking News Ticker** — Scrolling headlines across the top
- **Auto Refresh** — Updates every 30 minutes automatically

## 🛠️ Built With
- HTML, CSS, JavaScript
- Vite
- Groq AI API (Llama 3.3)
- Netlify (Hosting + Serverless Functions)
